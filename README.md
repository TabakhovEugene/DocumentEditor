## Инструкция по запуску

### 1. Клонирование репозитория
```bash
git clone [https://github.com/TabakhovEugene/DocumentEditor.git](https://github.com/TabakhovEugene/DocumentEditor.git)
cd DocumentEditor
```

### 2. Установка зависимостей

```bash
npm install
```

### 3. Запуск

```bash
npm run dev
```

После запуска терминал выведет локальный адрес (`http://localhost:5173/`). Откройте его в браузере для работы с приложением.
